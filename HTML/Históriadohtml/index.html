O básico do HTML ao HTML5

O que é o "HTML"?

HTML é uma abreviação da expressão inglesa "HyperText Markup Language",
que significa Linguagem de Marcação de Hipertexto. É uma linguagem de
marcação utilizada para produzir páginas da Web, documentos HTML que podem
ser interpretados por navegadores.

A criação do HTML

Em 1980, um físico britânico chamado Tim Berners-Lee, propôs um projeto
baseado no conceito de hipertexto para facilitar o compartilhamento e a
atualização de informações entre os pesquisadores, nascia aí o HTML. Sua
solução combinada com a internet pública ganhou a atenção mundial.

Etiquetas (tags)

Todo documento HTML possui tags, palavras entre < e >; essas tags são os comandos de formatação da linguagem.
Uma tag pode ou não possuir atributos e valores. Os atributos são utilizados para modificar o resultado padrão do elemento.

Edição de arquivo HTML

Os documentos em HTML são arquivos de texto simples que podem ser criados e editados em qualquer editor de textos comum, como o Bloco de Notas do Windows, ou o TextEdit, do Mac OS.

Parafacilitar a produção de documentos, no mercado existem editores de HTML específicos, com recursos sofisticados, que faculitam a realização de tarefas repetitivas, inserção de objetos, elaboração de tabelas e outros recursos, basicamente divide-se em dois tipos:

Editores de texto fonte
Inserem auutomaticamente as etiquetas, orientando a inserção de atributos e marcações, como o Notpad++
Editores WYSIWYG
Oferecem um ambiente deediçlão visual, com um "esboço" do resultado final, como o Dreamwaver

O Web Hypertext Application Tecnology Working Group (WHATWG) iniciou o trabalho do novo HTML em 2004, enquanto o World Wide Consotium (W3C) estava se concentrando no desenvolvimento do XHTML 2.0. Em 2009, o W3C decidiu que a equipe do XHTML 2.0 deveria parar seus trabalhos, e assim, descontinuar o padrão. Desta forma o W3C e o WHATWG passaram a trabalhar juntos no desenvolvimnto do HTML5.

Oprojeto HTML5 foi bem recebido pelos desenvolvedores Web até então, e tornou-setema da mídia em abril de 2010 depois que o CEO da Apple Inc., Steve Jobs emitiu uma carta pública intitulada "Reflexão sobre o Adobe Flash", onde ele concluía que o desenvolvimento do HTML5 tornaria o Adobe Flash não mais nescessário, tanto para assistir vídeos ou mesmo exibir qualquer conteúdo web.

Logo do HTML5

Em 18 de Jaeiro de 2011, a W3C introduziu um logo para representar o uso do HTML5. Diferente de outros emblemas anteriores disponibilizado pela W3C, esta não implica uma validação ou o uso correto da linguagem. Desde o dia 1º de abril, este logotipo é o oficial.
Segundo a W3C, o logotipo "Representa o HTML5, e a pedra angular para aplicações web modernas".
